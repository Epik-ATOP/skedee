How It Works Now

Each market page already has SM_HOME_NAME and SM_AWAY_NAME set. No fixture IDs needed — the page looks up odds automatically using the team names.

Odds are fetched once on page load to seed probabilities. During match window (±2 hrs around kick-off), live scores are polled every 30 seconds via smStartPoll(). The simulated match clock runs independently since The Odds API doesn't provide in-game events.
