After You Have the IDs

Open each market HTML file and find this line near the bottom:
const SM_FIXTURE_ID = null;

Replace null with the fixture ID number you copied, e.g.:
const SM_FIXTURE_ID = 19135821;

Save the file. The page will now automatically load real odds from Sportmonks when you open it, and switch to live polling when the match kicks off.
